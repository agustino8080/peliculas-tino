🔧 Instrucciones de Uso

Para usar en el reproductor original:

  1. Abre el archivo original: pelis.html
  2. Añade esta línea antes del cierre de </body>:
    <script src="subtitles.js"></script>
  3. Usa esta función para cargar subtítulos:
    loadSubtitlesFromM3U('https://github.com/agustino8080/listam3u/blob/main/comedia1tv.m3u')

Características del sistema:

  • ✅ Carga automática de listas M3U desde GitHub
  • ✅ Traducción de títulos de películas
  • ✅ Generación automática de subtítulos
  • ✅ Sincronización con el video
  • ✅ Soporte para múltiples idiomas
  • ✅ No modifica el HTML original
